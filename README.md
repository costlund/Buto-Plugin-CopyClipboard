# Buto-Plugin-CopyClipboard

<p>Copy content from an element with Javascript.</p>

<a name="key_0"></a>

## Usage



<a name="key_0_0"></a>

### Include

<pre><code>type: widget
data:
  plugin: copy/clipboard
  method: include</code></pre>

<a name="key_0_1"></a>

### Button

<pre><code>type: widget
data:
  plugin: copy/clipboard
  method: button
  data:
    id:  _id_of_element_
    text: Copy content</code></pre>
